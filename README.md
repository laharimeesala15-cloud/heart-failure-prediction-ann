# Heart Failure Prediction using ANN

Predicting heart failure risk from patient clinical data using an Artificial Neural Network.

## Dataset
- **Source:** Cardiovascular Disease Dataset — Indian Multispecialty Hospital
- **Link:** https://data.mendeley.com/datasets/dzz48mvjht/1
- **Records:** 1,000 patients | 12 clinical features

## Tech Stack
- Python 3
- TensorFlow / Keras
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn
- Google Colab

## Results
- Training Accuracy: 99.58%
- Validation Accuracy: 96.25%
